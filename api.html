Autenticação

Todas as requisições devem incluir o header Authorization: Bearer {seu_token}. Solicite seu token pelo WhatsApp.

GET https://api.kriar.digital/v2/me Verificar token 
# cURL
curl -X GET https://api.kriar.digital/v2/me \
  -H "Authorization: Bearer SEU_TOKEN_AQUI"

# Resposta:
{
  "id": "usr_abc123",
  "name": "Minha Empresa",
  "plan": "professional",
  "status": "active"
}

Enviar Mensagem

Envie mensagens de texto, imagens ou templates para contatos via WhatsApp, Instagram ou outros canais.

POST https://api.kriar.digital/v2/messages/send 
# Enviar mensagem de texto
curl -X POST https://api.kriar.digital/v2/messages/send \
  -H "Authorization: Bearer SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "channel": "whatsapp",
    "to": "5511999999999",
    "type": "text",
    "content": "Olá! Sua solicitação foi recebida."
  }'

Gerenciar Contatos

GET https://api.kriar.digital/v2/contacts Listar todos 
curl -X GET "https://api.kriar.digital/v2/contacts?page=1&limit=50" \
  -H "Authorization: Bearer SEU_TOKEN"
POST https://api.kriar.digital/v2/contacts Criar contato 
-d '{
  "name": "João Silva",
  "phone": "5511999999999",
  "email": "joao@exemplo.com",
  "tags": ["lead", "premium"]
}'

Webhooks Novo

Receba notificações em tempo real quando eventos ocorrem na plataforma.

POST https://api.kriar.digital/v2/webhooks 
-d '{
  "url": "https://seusite.com/webhook",
  "events": ["message.received", "conversation.closed"],
  "secret": "seu_segredo_hash"
}'
